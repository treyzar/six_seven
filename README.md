# Hand Tracker 6 & 7

Приложение для распознавания рук с отображением цифр 6 и 7.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python hand_tracker.py
```

## Управление

- Покажите руки перед камерой
- Левая рука = цифра 6 (розовый)
- Правая рука = цифра 7 (голубой)
- Нажмите `q` для выхода

## Возможности

- Распознавание до 2 рук одновременно
- MediaPipe для точного трекинга
- Светящиеся цифры с эффектом свечения
- Система частиц
- Плавная визуализация
