# Hand Tracker 6 & 7

Простое приложение: включает камеру, распознаёт руки и рисует цифры `6` и `7`.

Эта инструкция написана для человека **без опыта программирования**.

## Что понадобится

- Компьютер с Windows 10/11, macOS или Linux
- Интернет (для установки библиотек и загрузки модели)
- Веб-камера
- Python 3.10 или 3.11

Важно: на Python 3.12 `mediapipe` может не установиться.

## Шаг 1. Установите Python

1. Откройте сайт: `https://www.python.org/downloads/`
2. Скачайте Python 3.11.
3. Во время установки на Windows поставьте галочку `Add Python to PATH`.

## Шаг 2. Скачайте проект

Если проект уже у вас на компьютере, переходите к следующему шагу.

Если нет:

1. Установите Git: `https://git-scm.com/downloads`
2. Откройте терминал (PowerShell / Terminal).
3. Выполните команду:

```bash
git clone <ССЫЛКА_НА_ВАШ_РЕПОЗИТОРИЙ>
cd six_seven
```

## Шаг 3. Откройте папку проекта в терминале

В терминале нужно перейти в папку, где лежат файлы `hand_tracker.py` и `requirements.txt`.

Пример:

```bash
cd /путь/к/six_seven
```

## Шаг 4. Создайте виртуальное окружение (рекомендуется)

Это отдельная "мини-среда" для проекта, чтобы не ломать систему.

```bash
python -m venv .venv
```

Активация:

Windows (PowerShell):

```powershell
.venv\Scripts\Activate.ps1
```

Windows (cmd):

```cmd
.venv\Scripts\activate.bat
```

macOS/Linux:

```bash
source .venv/bin/activate
```

После активации слева в терминале обычно появляется `(.venv)`.

## Шаг 5. Установите зависимости

```bash
python -m pip install --upgrade pip
pip install -r requirements.txt
```

## Шаг 6. Скачайте файл модели

Проект использует модель MediaPipe: `hand_landmarker.task`.

По умолчанию приложение само попробует скачать файл при первом запуске.
Если интернет недоступен или автозагрузка не сработала, сделайте вручную:

1. Скачайте файл из официального репозитория MediaPipe:
   `https://storage.googleapis.com/mediapipe-models/hand_landmarker/hand_landmarker/float16/1/hand_landmarker.task`
2. Положите файл **в папку проекта**, рядом с `hand_tracker.py`.

Должно получиться так:

- `six_seven/hand_tracker.py`
- `six_seven/hand_landmarker.task`

Опционально: можно хранить модель в другом месте и задать переменную окружения:

```bash
HAND_LANDMARKER_MODEL=/полный/путь/hand_landmarker.task python hand_tracker.py
```

## Шаг 7. Запуск

```bash
python hand_tracker.py
```

Откроется окно с камерой.

Управление:

- Покажите руки перед камерой
- Для комбо не нужно удерживать руки
- Нужно один раз "коснуться" уровней: одна рука до линии `TOP`, другая до линии `BOTTOM`
- Работают оба варианта: левая сверху/правая снизу и наоборот
- Чтобы засчиталось следующее комбо, вернитесь из позы касания и снова коснитесь уровней
- Нажмите `r`, чтобы сбросить комбо
- Нажмите `q`, чтобы выйти

## Частые проблемы и решение

`ModuleNotFoundError`:

- Вы не установили зависимости или не активировали окружение.
- Снова активируйте `.venv` и выполните `pip install -r requirements.txt`.

`No module named cv2`:

- Не установлен `opencv-python`.
- Выполните `pip install opencv-python`.

Камера не открывается:

- Закройте другие приложения, которые используют камеру (Zoom, Telegram и т.д.).
- Проверьте разрешение на доступ к камере в настройках ОС.

Ошибка про `hand_landmarker.task`:

- Модель не скачана или лежит не в той папке.
- Проверьте, что файл лежит рядом с `hand_tracker.py`.

## Как остановить проект

1. Нажмите `q` в окне камеры.
2. В терминале (если нужно) нажмите `Ctrl + C`.

## Что делает программа

- Определяет до 2 рук одновременно
- Рисует точки и линии скелета кисти
- Показывает большие светящиеся цифры
- Добавляет простой визуальный эффект частиц
- Добавляет комбо-механику `6 + 7` с анализатором уровней `TOP/BOTTOM` (без удержания)
- На высоких сериях включает усиленные эффекты: шлейф движения, flash и `SUPER 67`
- Пасхалка: при `67` комбо запускается большой взрыв конфетти
